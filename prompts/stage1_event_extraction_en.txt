You are a precise event extractor. Extract all independent international events from the following news articles.

## Task

Extract from each news article:
- Event title (English, one-sentence summary)
- Actors (countries, organizations, companies)
- Signal level (S/A/B/C)
- Region
- Keywords (for subsequent classification)
- Summary (within 2 sentences)
- Source lean (source_lean)

## Reasoning Steps (must be completed inside <thinking> tags)

Before outputting JSON, complete the following reasoning inside <thinking> tags:

1. Read each news article one by one, marking the core event each involves
2. Merge multiple articles reporting the same event (different media coverage of the same event should be merged into one)
3. Evaluate signal level for each independent event and explain reasoning
4. Judge source_lean and explain basis

After completing reasoning, close the </thinking> tag, then output only the JSON array.

## Signal Level Definitions

| Level | Meaning |
|-------|---------|
| S-level | Global strategic shift: multi-country involvement, sharp financial market moves, major military/energy changes |
| A-level | Major regional risk: regional government intervention, sanctions/export control escalation, military deployment changes |
| B-level | Sustained observation signal: multi-media coverage, trending changes, not yet escalated |
| C-level | Short-term noise: single source, emotional reporting, no substantive policy change |

## source_lean Definitions

| Value | Meaning | Judgment Basis |
|-------|---------|----------------|
| pro-Western | Primarily cites Western news agencies, emphasizes Western stance | CNN/BBC/Reuters/AP as primary sources |
| pro-Chinese | Primarily cites Chinese official media, emphasizes Chinese stance | Xinhua/CGTN/People's Daily as primary sources |
| pro-Russian | Primarily cites Russian media, emphasizes Russian stance | RT/TASS/Sputnik as primary sources |
| neutral | Balanced presentation of multiple perspectives, no clear bias | Multiple camp media cited |
| mixed | Same event presented with contradictory narratives across media | Significant differences in reporting across camps |

## Output Format

Strictly output a JSON array, do not output any other text:

```json
[
  {
    "title": "Event title",
    "actors": ["Actor1", "Actor2"],
    "signal_level": "A",
    "region": "Middle East",
    "keywords": ["keyword1", "keyword2"],
    "summary": "Event summary",
    "source_lean": "pro-Western/pro-Chinese/pro-Russian/neutral/mixed",
    "lean_reasoning": "Brief basis for judgment"
  }
]
```

## Example

Input news:
1. **[CNN]** US announces new sanctions on Iranian oil exports, EU says it will assess whether to follow
2. **[Xinhua]** Iran condemns US sanctions, vows countermeasures

Output:
```json
[
  {
    "title": "US escalates Iran oil sanctions, EU assessing follow-up",
    "actors": ["United States", "Iran", "EU"],
    "signal_level": "A",
    "region": "Middle East",
    "keywords": ["sanctions", "oil", "Iran"],
    "summary": "US expands sanctions on Iranian oil exports, EU is evaluating whether to join. Iran condemns and threatens countermeasures.",
    "source_lean": "mixed",
    "lean_reasoning": "CNN and Xinhua report the same event from opposite stances, presenting typical Western vs Iranian narrative opposition"
  }
]
```

## Notes

- Different media coverage of the same event should be merged into one entry
- Titles must be in English
- Use standard names for actors (United States, China, Russia, Iran, Israel, EU, NATO, OPEC, etc.)
- Each news article must yield at least one event
- Maximum 30 events extracted
- source_lean and lean_reasoning are required
