- **100+ global sources** across 12 regions
- **Multi-source narrative comparison** — how different camps frame the same event

- **Event evolution tracking** — diplomatic → military → financial phases
- **Real market data** — US/HK/A-shares, commodities, FX, treasuries
- **90-day historical memory** — AI connects today to yesterday
- **Email + Telegram delivery**

## Quick Start

```bash
git clone https://github.com/vanthree31/PrismLens.git
cd PrismLens
pip install -r requirements.txt
cp .env.example .env   # Edit with your API key
python run.py           # Generate today's briefing
```

**Requirements:** Python 3.10+, DeepSeek API key (or any OpenAI-compatible API).

See [`.env.example`](.env.example) for all options.

## How It Works

```
Scheduled trigger (daily)
        │
        ▼
Fetch 100+ RSS feeds in parallel
        │
        ▼
AI reads everything in one pass — all articles,
yesterday's events, market data, 90-day trends
        │
        ▼
Outputs complete Markdown report + JSON metadata
        │
        ▼
Render HTML → Email → Telegram → Archive
```

No fragmented pipeline. No intermediate parsing failures. The AI sees the full picture.
