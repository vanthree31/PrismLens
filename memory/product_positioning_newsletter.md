# PrismLens 产品定位

> **PrismLens = AI 国际局势日报 Newsletter 服务**
>
> 不是 SaaS 平台。不是 OSINT 工具。不是给别人装软件。

## 用户流程

```
服务器定时抓取 111 RSS → AI 生成日报 → 推送给用户
```

用户只看到：邮箱 / Telegram / 网页日报。结束。

## 核心竞争力

1. 日报质量（AI 分析深度、叙事对比、风险评估）
2. AI Memory（Event Database 让 AI 拥有跨日长期记忆，日报越来越聪明）
3. 推送体验（邮件、Telegram 等渠道稳定可靠）
4. 付费转化（免费版看到质量 → Pro 订阅）

## 用户永远不会做的事

- 登录后台
- 点击 Timeline
- 查询 Event Database
- 配置 Dashboard
- 搜索事件

这些是给 AI 用的基础设施，不是给用户的功能。

## Event Database 的真正用途

**给 AI 当长期记忆。** 不是给用户查询。AI 写日报时自动读取相关事件的历史状态，写出"这是该事件第 12 天处于军事阶段"这种有历史深度的分析。

## 前端只需要三个页面

1. 今日日报（阅读全文）
2. 历史日报（按日期归档，点击阅读）
3. 订阅页面（输入邮箱 → Subscribe）

## 参考产品形态

Morning Brew / The Information / The Hustle — 但它们写商业新闻，PrismLens 写国际战略情报。
